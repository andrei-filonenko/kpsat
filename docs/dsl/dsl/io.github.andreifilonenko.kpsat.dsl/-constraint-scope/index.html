ConstraintScope

interface ConstraintScope

Scope for building constraint programming models. Provides DSL methods for creating variables, arrays, and emitting constraints.

Used as a context receiver:

context(ConstraintScope)
fun buildModel(vars: MutableMap<String, Expr>) {
    vars["x"] = int(0, 100)
    hard("positive") { vars["x"]!! geq 0L }
}
