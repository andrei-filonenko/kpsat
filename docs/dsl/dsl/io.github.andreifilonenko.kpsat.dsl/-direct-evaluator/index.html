DirectEvaluator

class DirectEvaluator

Direct evaluator for expressions without a solver. Useful for unit testing constraint logic.

Usage:

val evaluator = DirectEvaluator()
    .withVar(varId, 5L)
    .withVar(varId2, 10L)
val result = evaluator.evaluate(expr)
