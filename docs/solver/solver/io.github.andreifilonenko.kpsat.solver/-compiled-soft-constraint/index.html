CompiledSoftConstraint

data class CompiledSoftConstraint(val name: String, val penaltyVar: IntVar, val weight: Int, val priority: Int, val minBound: Long, val maxBound: Long)

Compiled soft constraint ready for optimization.

Soft constraints at the same priority level are normalized to ensure fair comparison. The normalization follows OptaPlanner's approach:

  1. Estimate min/max bounds for each constraint

  2. Scale penalties to a common range 0, NORMALIZATION_SCALE

  3. Apply weights after normalization

  4. Sum normalized weighted penalties per priority level
