ENodeOp

sealed interface ENodeOp

E-graph node operation. This is a simplified representation of ExprNode where children are replaced by EClassId references.

IMPORTANT: This sealed interface must have a variant for every ExprNode variant. The compiler enforces exhaustive when expressions.
